# Source Mockups

Upload the full approved Paper Tiger section mockups here.

These files are the **primary visual source** for implementation when coding the landing page. They show the complete intended composition of each section and should guide the reconstruction of the live website.

## Critical implementation rule

These mockups are **not** intended to be inserted into the website as full-screen section images.

Codex / the implementer must use them to:

- understand exact layout, proportions, and visual hierarchy;
- recreate the sections as responsive HTML/CSS plus production image assets;
- decide what should remain real text, real forms, and real interface elements;
- isolate, mask, crop, or derive artwork from the mockups only where that is genuinely useful;
- remove or recreate non-production backgrounds when needed;
- avoid relying on fragmented loose elements when those create confusion.

Do **not** solve implementation by stacking the uploaded mockup images as flat page screens.
